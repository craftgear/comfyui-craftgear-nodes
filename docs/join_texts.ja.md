# Join Texts

## 特徴

- 複数のテキスト入力を1つのテキストに結合します。
- 入力ソケットは動的に追加されます。接続すると新しい入力が自動的に作成されます。
- 区切り文字の重複を自動的に処理します。


## 入力パラメータ

| パラメータ | 型 | 説明 |
|-----------|------|------|
| text_1 | STRING | 最初のテキスト入力 (必須、ソケット入力) |
| separator | STRING | テキスト結合に使用する区切り文字 (デフォルト: `,`) |
| text_2, text_3, ... | STRING | 追加のテキスト入力 (接続時に動的に追加) |


## 出力

| 出力 | 型 | 説明 |
|------|------|------|
| text | STRING | 区切り文字で結合されたテキスト |


## 動的入力

- `text_1` に接続すると、自動的に `text_2` の入力ソケットが追加されます。
- 接続を続けると `text_3`, `text_4`, ... と入力が増えていきます。


## 区切り文字の処理

入力テキストの末尾や先頭に区切り文字が既に含まれている場合、重複を防ぎます:

- 入力1: `tag1, tag2,`
- 入力2: `, tag3, tag4`
- 出力: `tag1, tag2, tag3, tag4`


## 使用例

複数のノードから出力されたタグやプロンプトを1つにまとめる場合に使用します。Load LoRAs With Tags の tags 出力と他のプロンプトノードを結合するのに便利です。
