# Toggle Tags

## 特徴

- 入力されたタグテキストを表示し、クリックでタグのオン/オフを切り替えられます。
- 無効化されたタグは取り消し線で表示され、出力から除外されます。
- ワークフロー実行中にタグの有効/無効を素早く切り替えることができます。


## 入力パラメータ

| パラメータ | 型 | 説明 |
|-----------|------|------|
| text | STRING | カンマ区切りのタグテキスト (ソケット入力必須) |


## 出力

| 出力 | 型 | 説明 |
|------|------|------|
| text | STRING | 有効なタグのみをカンマ区切りで結合したテキスト |


## 使い方

1. タグテキストを `text` 入力に接続します。
2. ノード内に表示されたタグをクリックすると、そのタグが無効化されます。
3. 無効化されたタグは取り消し線付きで表示され、出力には含まれません。
4. 再度クリックすると有効に戻ります。


## 表示について

- 有効なタグ: 通常表示
- 無効なタグ: 取り消し線 + 半透明表示
- タグが入力されていない場合: "(empty)" と表示


## 使用例

Load LoRAs With Tags の tags 出力を接続し、生成時に使用するタグを動的に選択する場合に便利です。複数のタグの中から必要なものだけを有効にして生成できます。
