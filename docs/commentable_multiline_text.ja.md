# Commentable Multiline Text

![node](./images/commentable_multi_line.png)

## 特徴

- 複数行のテキストからコメント行を除去して出力します。
- `#` または `//` で始まる行はコメントとして扱われ、出力から除外されます。
- プロンプトの一部を一時的に無効化したい場合に便利です。


## 入力パラメータ

| パラメータ | 型 | 説明 |
|-----------|------|------|
| text | STRING | 複数行のテキスト入力 |
| separator | STRING | 行の結合に使用する区切り文字 (デフォルト: `,`) |


## 出力

| 出力 | 型 | 説明 |
|------|------|------|
| text | STRING | コメント行を除去し、区切り文字で結合されたテキスト |


## コメント記法

以下の形式でコメントを記述できます:

```
# これはコメントです (出力されません)
// これもコメントです (出力されません)
有効なテキスト
  # 行頭の空白の後でもコメントとして認識されます
```


## 使用例

プロンプトを整理しながら、一部のタグを一時的に無効化したい場合に使用します:

```
1girl
blue eyes
# red hair
blonde hair
// test tag
```

上記の入力は `1girl, blue eyes, blonde hair` として出力されます。
