# Image Batch Loader

![node](./images/image_batch_loader.png)

## 特徴

- 指定したディレクトリ内の画像ファイルをバッチとして読み込みます。
- 正規表現によるファイル名フィルタリングが可能です。
- 同じサイズの画像のみがバッチに含まれます。


## 入力パラメータ

| パラメータ | 型 | 説明 |
|-----------|------|------|
| directory | STRING | 画像ファイルが格納されているディレクトリのパス |
| filter | STRING | ファイル名をフィルタリングする正規表現パターン (空の場合は全ファイル) |


## 出力

| 出力 | 型 | 説明 |
|------|------|------|
| batch | IMAGE | 読み込まれた画像のバッチ |


## 対応フォーマット

- PNG (.png)
- JPEG (.jpg, .jpeg)
- WebP (.webp)


## フィルタパターンの例

| パターン | 説明 |
|----------|------|
| `frame` | ファイル名に "frame" を含むファイル |
| `^img_` | ファイル名が "img_" で始まるファイル |
| `\d{4}` | ファイル名に4桁の数字を含むファイル |
| `\.png$` | PNG ファイルのみ |


## サイズの統一

バッチ処理のため、最初に読み込まれた画像と同じサイズの画像のみがバッチに含まれます。異なるサイズの画像は自動的にスキップされます。


## 使用例

連番画像を動画として処理したい場合や、複数の画像に同じ処理を適用したい場合に使用します。
